Projet d’automatisation des tests ornitho.com

![Playwright](https://img.shields.io/badge/Powered%20by-Playwright-blue.svg)

Ce projet est pour tester le site web d’Ornitho, une plateforme pour les rapports sur l’observation des oiseaux. 
Construit avec Playwright (TypeScript).

Prérequis : 

> Node.js (v18 ou supérieur) 
> Git 
> Un éditeur de code comme VS Code (recommandé pour les extensions Playwright)
